- FAST 모드 안내 메시지를 보강해 학습 후 반드시 검증 셀(#28)을 실행해야 함을 명시했습니다 (`day_03/어댑터로 돌리기/a100_1.ipynb:1`).
- 검증/추론 앙상블은 원본 밝기를 항상 포함하고, 자원 한계에 맞춰 밝기 수준을 자동 조정하도록 개선했습니다 (`day_03/어댑터로 돌리기/a100_1.ipynb:1`).

## 남아있는 주의사항
- HF 토큰이 다시 코드에 하드코딩돼 있으므로 저장소 공유나 외부 배포 시 반드시 토큰을 제거하거나 무효화해야 합니다 (`day_03/어댑터로 돌리기/a100_1.ipynb:1`).
- 앙상블은 여전히 배치당 여러 이미지를 생성하므로 VRAM이 부족하면 `ENSEMBLE_MAX_IMAGES`나 `ENSEMBLE_BRIGHTNESS_LEVELS`를 더 낮춰야 합니다.
- FAST 모드에서는 검증 샘플 수가 적어 지표 변동이 큽니다. 제출 전에 FAST 모드를 끄고 전체 데이터로 학습과 검증을 다시 수행해야 합니다.
- 이미지 캐시가 장시간 학습 시 메모리를 차지할 수 있으므로, 필요하다면 캐시를 비우는 헬퍼를 추가하는 방안을 검토하세요.
